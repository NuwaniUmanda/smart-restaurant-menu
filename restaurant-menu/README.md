# Restaurant Menu System

A web-based restaurant menu system with QR code access, AI chatbot (Dialogflow), Firebase backend, and modern responsive UI using React and Tailwind CSS.

## Tech Stack
- React (JavaScript)
- Tailwind CSS
- Firebase (Firestore)
- Google Dialogflow
- Lucide React Icons

## Features
- Welcome, Menu, and Cart pages
- AI-powered chatbot for recommendations
- Real-time order and menu management
- Mobile-first, responsive design
